# Flower Classification

A beginner-friendly Machine Learning project to classify Iris flowers into 3 species using Random Forest Classifier.

## About
The Iris dataset contains 150 samples of 3 flower species:
- Setosa
- Versicolor  
- Virginica

## Libraries Used
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Model
- **Algorithm:** Random Forest Classifier
- **Accuracy:** ~96-100%
- **Train/Test Split:** 80/20
